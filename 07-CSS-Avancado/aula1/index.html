Variables

Definição:

Variables, como o próprio nome diz, são as variáveis do CSS, e são como as variáveis de qualquer linguagem, você armazena um valor dentro de um nome para reutilizá-lo depois.

Criação:

Normalmente é introduzida dentro do seletor root ( :root {} ), para ser acessível a todo o documento, mas pode ser utilizada dentro do escopo de qualquer seletor, e seu nome sempre deve ser iniciado por "--" da seguinte forma:

:root {
  --cor-primaria: red;
}

Utilização:

Para ser utilizada a variable deve ser estar dentro de parenteses atrelados à palavra "var" da seguinte forma:

var(--cor-primaria)

Ela é utilizada como o valor de uma propriedade de um seletor CSS. Exemplo:

:root {
  --cor-primaria: red;
}

h1 {
  color: var(--cor-primaria);
  border-color: var(--cor-primaria)
}

Import

Definição:

É um elemento que adiciona escalabilidade e praticidade à configuração visual de um site com muitas páginas diferentes. Ela serve para diminuir a repetição no trabalho de edição em caso de alteração.

Utilização:

Para utilizar o import basta adicionar um @ seguido da palavra import, um espaço simples e o url completo do documento css que se deseja importar, por exemplo:

@import 'arquivo-a-ser-importado.css'
